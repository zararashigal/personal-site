HOW DOES IT WORK?

If you want to search for a lobby:
1. Start System Shock 2 Anniversary
2. Click Multiplayer
3. Click Online
4. Click Join Code
5. In SS2MS, click Search for a Lobby
6. Give it a bit...
7. Ingame, copy the Room Code you have received into the textbox.
8. Click join.
If you want to host a lobby for others to search for:
1. Start System Shock 2 Anniversary
2. Click Multiplayer
3. Click Online
4. Click Host Game
5. Choose a save or a new session
6. Copy the Room Code in the lower right into the textfield.
7. Click Add to Pool.
NOTE: Re-add your room code to the lobby pool after about 2 minutes, lobby instances decay to avoid abuse.
